# 股市分析面板 — 設計規劃文件

> 即時追蹤美股與台股消息面，並判斷對關注個股的影響
> 版本 v1.0｜日期 2026-06-07｜成品形式：規劃文件（後續可依此實作互動面板）

---

## 1. 專案目標

打造一個**單頁、可重複開啟、自動刷新**的股市分析面板，協助使用者在開盤前後快速掌握三件事：

1. **美股與台股的消息面**現在發生了什麼（新聞情緒：利多／利空）。
2. **關注個股**的最新報價、漲跌幅與成交量。
3. **大盤與總經指標**的方向（指數、匯率、利率、VIX 等）。

核心價值在於**把「新聞」與「個股」連起來**——不只是列新聞，而是標示「這則消息影響哪一檔股票、偏多還偏空」。

### 重要前提：即時 vs 準即時

真正的逐筆即時報價（tick-by-tick）需要**付費資料源**（券商 API、富果、富邦、彭博、Refinitiv）。本面板在無付費源的前提下，設計為**準即時（near real-time）**：

| 資料類型 | 可達到的更新頻率 | 說明 |
|---|---|---|
| 美股報價 | 即時～延遲 15 分 | 免費源多為延遲報價；即時需付費 |
| 台股報價 | 延遲（證交所 OpenAPI 為前一日／盤後快照為主） | 盤中即時需富果／富邦付費 API |
| 新聞消息 | 準即時（數分鐘） | 透過新聞 API 或網路搜尋抓取 |
| 總經指標 | 每日／每次發布 | 利率、CPI 等屬低頻資料 |

面板會在每個數字旁標示**資料時間戳與延遲狀態**，避免誤判為逐筆即時。

## 3. 系統架構總覽

```
┌─────────────────────────────────────────────────────────┐
│                     股市分析面板（單頁 UI）                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────────┐ │
│  │ 大盤總經  │ │ 關注個股  │ │ 新聞情緒  │ │ 個股影響映射  │ │
│  │  指標列   │ │  報價牆   │ │   動態流  │ │   （連動圖）   │ │
│  └──────────┘ └──────────┘ └──────────┘ └──────────────┘ │
└───────────────────────────┬─────────────────────────────┘
                            │ 每次開啟／點刷新時拉取
        ┌───────────────────┼───────────────────┐
        ▼                   ▼                   ▼
   ┌─────────┐         ┌─────────┐         ┌──────────┐
   │ 報價資料源 │         │ 新聞資料源 │         │ 總經資料源 │
   │ 美股/台股 │         │ 美/台新聞 │         │ 利率/匯率 │
   └─────────┘         └────┬────┘         └──────────┘
                            ▼
                   ┌─────────────────┐
                   │ 情緒分析 + 個股映射 │
                   │ (NLP 判斷利多/利空) │
                   └─────────────────┘
```

設計原則：**資料在開啟頁面時即時拉取（透過連接器或新聞 API）**，並可手動點「刷新」重抓，不需自建後端伺服器即可運作。

## 4. 資料源規劃

### 4.1 美股報價

| 資料源 | 免費額度 | 即時性 | 建議用途 |
|---|---|---|---|
| **Finnhub** | 60 次/分鐘 | 即時 US 報價（免費含基本即時） | **首選**——報價＋公司新聞＋情緒一站式 |
| Alpha Vantage | 25 次/日（免費） | 延遲 | 備援、技術指標、新聞情緒 |

| Twelve Data / Marketstack | 有限免費 | 延遲 | 多市場報價備援 |

> Yahoo Finance 官方 API 已停用，非官方爬取不穩定，**不建議作為主力**。

### 4.2 台股報價

| 資料源 | 性質 | 即時性 | 備註 |
|---|---|---|---|
| **證交所 OpenAPI**（openapi.twse.com.tw） | 官方免費 | 盤後／前一日為主 | 速率限制嚴格 |
| 證交所 MIS（mis.twse.com.tw getStockInfo） | 官方免費 | 盤中延遲 | 每 5 秒最多 3 次請求 |
| 櫃買中心 OpenAPI（tpex.org.tw） | 官方免費 | 盤後 | 上櫃／興櫃 |
| **富果 Fugle API** | 免費＋付費 | **盤中即時** | 真正即時需註冊／付費 |

> **建議**：MVP 階段用證交所 OpenAPI（免費、合法、穩定）做盤後／延遲報價；若要盤中即時，升級接富果或富邦。

### 4.3 新聞消息面

| 資料源 | 涵蓋 | 情緒分析 | 備註 |
|---|---|---|---|
| **Finnhub News + Sentiment** | 美股公司新聞 | 內建 sentiment 分數 | 免費，含個股關聯 |

> **台股中文新聞無成熟免費 API**，建議以「網路搜尋 + AI 情緒判讀」為主要管道，這也是本面板能跨語言（中／英）統一處理的優勢。

### 配色與標示規則
- 🟢 綠／紅 🔴：依**台股慣例**可選「漲紅跌綠」，或國際慣例「漲綠跌紅」（面板提供切換）。
- 每張卡片右下角顯示**資料時間戳 + 延遲標籤**（即時／延遲 15 分／盤後）。
- 新聞依情緒著色邊框：利多綠、利空紅、中性灰。

## 6. 核心功能模組規格

### 模組 ①：大盤與總經指標帶
- **內容**：美股三大指數、SOX、TAIEX、USD/TWD、US10Y、VIX。
- **呈現**：橫向卡片，每張含數值、漲跌幅、方向箭頭、迷你趨勢線。
- **更新**：開啟頁面即抓；指數延遲報價可接受。

### 模組 ②：關注個股報價牆
- **內容**：每檔顯示 代號／名稱／現價／漲跌幅／成交量／當日迷你 K 線。
- **分組**：依「美股科技權值 / 台股半導體 / AI 供應鏈」三區塊。
- **互動**：點個股 → 展開該檔近期相關新聞與情緒。
- **延遲標示**：台股盤中為延遲，美股為即時／延遲視資料源。

### 模組 ③：新聞情緒動態流
- **內容**：時間倒序的新聞卡片，每則含：
  - 情緒標籤（利多🟢／利空🔴／中性⚪）
  - 一句話摘要
  - **受影響個股**（自動標註 ticker）
  - 來源與時間
- **來源**：Finnhub 美股新聞 + 網路搜尋（台股中文新聞）。
- **情緒判斷**：見第 7 節邏輯。

### 模組 ④：個股影響映射（連動圖）
- **目的**：把一則消息的**傳導鏈**畫出來。
- **範例**：「NVDA 財報優 → 上游晶片利多 → 台積電、廣達、緯創（供應鏈）同步偏多」。
- **呈現**：簡單的節點連線圖或縮排樹狀清單。

## 7. 新聞 → 個股影響判斷邏輯

這是面板的「大腦」，分三步：

### 步驟 1：情緒分類
對每則新聞判定 **利多 / 利空 / 中性**：
- 優先採用資料源內建情緒分數（Finnhub、Alpha Vantage）。
- 中文新聞或無分數者，由 AI 即時判讀（關鍵詞 + 語意）。

### 步驟 2：個股關聯
判定這則消息影響哪些 ticker：
- **直接關聯**：新聞直接點名的公司。
- **供應鏈關聯**：套用預建的「關聯映射表」（見下）。

### 步驟 3：方向傳導
依關聯類型決定方向：

| 事件類型 | 直接標的 | 傳導標的（範例） |
|---|---|---|
| NVDA 財報／AI 需求利多 | NVDA 🟢 | 台積電🟢、廣達🟢、緯創🟢、台達電🟢 |
| 美國升息／鷹派 | 大盤🔴 | 科技權值偏空、金融偏多 |
| 台積電擴產／法說利多 | 2330🟢 | 設備、封測、供應鏈🟢 |
| 美對中半導體管制 | 視個股 | 部分台廠轉單利多／部分利空 |
| 蘋果新品／拉貨 | AAPL🟢 | 鴻海🟢、台積電🟢、台達電🟢 |

> **關聯映射表**為一份可維護的設定檔（公司 ↔ 供應鏈 ↔ 客戶關係），面板初期手動建立核心 20–30 條關係，之後逐步擴充。

## 8. 自動刷新與排程

| 機制 | 說明 |
|---|---|
| **開啟即抓** | 每次打開面板自動拉取最新資料（已內建快取避免過度請求） |
| **手動刷新** | 面板頂部一鍵重抓 |
| **每日定時推送**（可選） | 可設定排程，例如每日台股開盤前 08:30、美股收盤後 05:00 自動產生「盤前／盤後摘要」 |

> 若要每日自動摘要，可另外設定排程任務，每天固定時間產生一份當日消息面 + 個股影響的摘要。

## 10. 風險與限制

1. **免費源即時性有限**：台股盤中、美股即時報價在免費方案下多為延遲；面板已用時間戳明示，但決策前須注意。
2. **速率限制**：證交所 5 秒 3 次、Alpha Vantage 每日 25 次等，需快取與節流。
3. **台股中文新聞無成熟 API**：仰賴網路搜尋，覆蓋度與時效視搜尋結果而定。
4. **情緒判斷非投資建議**：利多／利空為輔助參考，不構成買賣建議。
5. **供應鏈傳導為機率性推論**：實際個股反應受多重因素影響，映射表僅供方向參考。
